# Best USA Camps

WordPress plugin and theme templates for managing summer camp profiles, user dashboards, and custom login flows.

## Key Features
- Camp dashboard for camp directors (profile, uploads, activities)
- Custom login, lost password, and reset password pages (Astra theme)
- Storage tracking for photos and logo uploads
- Ninja Forms integration for camp registration
